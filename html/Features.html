App Features:

While the features of the Oratryx will still need to be decided upon, the developers will utilize the following features:

  • User Accounts: Users can create and access their accounts by signing up and logging in.
  • User Calendar: Users can choose their own time and date.
  • Dental Dentist Profile: Users can see the profile of the dentist they want to work with.
  • User/Patient History: The user can view their history of transactions with that dental clinic.
  • Services Details and Prices: Users can check the services that are offered by the dental clinic in the application.
  • OTP Verification: Before successfully approving the schedule, users must verify their accounts.
  • Feedback Section: Users can review the service that they experienced from the dental clinic.
